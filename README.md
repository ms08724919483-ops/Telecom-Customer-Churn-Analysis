# 📊 Telecom Customer Churn Analysis & Retention Strategy Dashboard

## 📌 Project Overview
This project analyzes 7,000 telecom customers to identify churn drivers, quantify revenue risk, and simulate retention strategy impact using SQL, Excel, and Power BI.

---

## 🔍 Key Business Metrics

- Total Customers: 7,000
- Churn Rate: 47.44%
- Total Revenue: ₹17.91M
- Revenue Lost from Churn: ₹8.21M
- Avg Revenue per Churned Customer: ₹2,474

---

## 🚨 Key Insights

- Month-to-Month contracts drive majority of churn revenue loss (~₹6.35M)
- Fiber Optic customers show highest churn rate (~61%)
- Customers without Tech Support churn 53% vs 32% with support
- 25% retention improvement could recover ~₹1.59M revenue

---

## 🛠 Tools Used

- SQL (Joins, CTEs, Aggregations)
- Power BI (DAX, Data Modeling, KPI Dashboard)
- Excel (Data Cleaning & Validation)
- Python (Pandas – basic data analysis)

---

## 📈 Dashboard Pages

1. Executive Overview

---

## 🎯 Business Outcome

Developed a data-driven retention strategy highlighting contract migration and service optimization to reduce churn-related revenue loss.
